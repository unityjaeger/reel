Reel evaluates `KeyframeSequence` pose data into local `Motor6D.Transform` or `Bone.Transform` values.
It can also reconstruct those results in world space for server-side queries such as lag compensation.

It is a solver, not an animation-player replacement.
Your code owns track lifecycle, time progression, fading, replication, and committing results to Instances.
